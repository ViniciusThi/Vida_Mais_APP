## Como usar este arquivo

Cada entrada segue o formato:

```
### [YYYY-MM-DD] Título curto do problema
**Contexto:** onde aconteceu e o que causou
**Sintoma:** mensagem de erro ou comportamento inesperado
**Causa raiz:** o que estava errado de verdade
**Solução:** o que resolveu
**Lição:** o que aprender para não repetir
```
